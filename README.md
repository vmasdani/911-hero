# 911 Hero

Superhero dispatching game, fully on browser.

### Running

```sh
yarn dev
```

### Building

```sh
yarn build
```

[Demo](https://911hero.vmasdani.masuk.id)

Assets used:

- [Hero portraits (from itch.io)](https://loregret.itch.io/pixel-art-portrait-pack?download)
- [BGM by soundridemusic](https://www.youtube.com/watch?v=BEVg8Chp9OE)
